Lesson 1️⃣: Dictionaries

Bitesize Key points from this lesson

  1. This lesson was focused on dictionaries in Python, including how to create, access, and modify them.
  2. We learned about the syntax for creating dictionaries and how to add items using update(), and remove key-value pairs using pop().
  3. We explored common dictionary methods such as keys(), values(), and how to interact with dictionaries.
  4. We also learned about nested dictionaries and how a dictionary can contain other dictionaries as values.
  5. Lastly, we added a new skill to our knowledge in functions by learning how to pass optional parameters.
